学习单片机四要点

  • 理解

    学会C语言的常用语法，理解51单片机开发平台软件keil的操作，理解单片机的原理特性，能读懂按键，数码管，跑马灯，串口的简单程序，能熟悉几个常用的外围芯片驱动。

  • 借鉴

    当开始接触一些小项目时，往往不知道从何处入手，这是因为缺少程序的框架思路。所以需要通过与他人进行探讨交流以及学习和借鉴他人的分享经验来摸索开发项目的门径。

  • 掌握

    在做项目时，除了写单片机的软件，还不可避免的要跟整个电路的硬件打交道，所以掌握一些常用的硬件电路知识就显得尤其重要。

  • 实践

    唯有通过大量的实践才能更牢固地掌握知识，积累开发项目的经验，发现自身知识部分的不足，从而更深入地研究该领域所需知识。

  • 理解
  • 借鉴
  • 掌握
  • 实践
